Zara Sales Performance Dashboard

Business Intelligence Final Project

Project Overview
================

This project aims at demonstrating an interactive BI dashboard that focuses on the sales performance of Zara from 2013 to 2018. The main purpose is to be able to interpret the sales information in a manner that supports smart decision-making in the clothing retailing industry that is associated with fast fashion.
Company Context

Zara operates in fast fashion, where demand fluctuates quickly with a need to make quick decisions on inventory. Through analysis of previous sales, Zara can:
- Top products and best-performing stores

- Identify spot sales trends and seasonal patterns

- Assist in planning inventory & operations

Business Question
================
In what way can Zara use sales information in order to detect trends, bestselling goods, as well as store performance?
Important Dashboard Features
- Executive Dashboard and KPI Summary
Interactive charts (line, bar, scatter, donut)
• Store Performance Cluster Analysis
- Top-Selling Product Analysis

- Seasonality Insights
- Clear business insights and recommendations
Storytelling to describe why research findings are important

-- All charts are interactive: hover, zoom, and explore
